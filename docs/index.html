OpenOS

A modular, research-oriented operating system built from scratch for the x86 architecture — featuring memory management, exception handling, and a built-in shell.

View on GitHub Source Code Contributing

Project Status

Phase 0 — Core Foundation complete

Key Features

🧱

Modular Monolithic Kernel

Clean separation between arch, kernel, memory, drivers, and filesystem layers.

⚡

Exception Handling

All 32 x86 CPU exceptions handled with detailed crash reports and register dumps.

🧠

Memory Management

Bitmap-based physical memory manager and two-level virtual memory paging.

⌨️

Built-in Shell

Interactive shell with ls, cat, stat, echo, and more.

🔬

CPU Simulator

5-stage pipelined CPU, single-cycle reference model, cache, and bus simulator.

🚀

QEMU & VirtualBox

Run directly in QEMU or boot a full ISO in VirtualBox with one command.

Repository Sections

  • kernel/ — Core kernel initialization and panic handling
  • arch/ — x86 architecture code (IDT, ISR, PIC, exceptions)
  • memory/ — Physical and virtual memory managers
  • drivers/ — Console (VGA), keyboard (PS/2), timer (PIT)
  • docs/ — Architecture docs, roadmap, and learning materials
  • Makefile — Build, run, ISO, benchmark, debug targets
  • CONTRIBUTING.md — How to contribute

Quick Start

  • Build & run in QEMU (make run)
  • Build a bootable ISO (make iso)
